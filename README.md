# 📱 Smart Telecom Complaints System (نظام إدارة شكاوى الاتصالات الذكي)

## 🚀 About The Project
A comprehensive mobile application and management dashboard designed to streamline the process of submitting, tracking, and resolving telecom complaints. Powered by an AI engine for automated categorization.

## 🛠️ Tech Stack
* **Frontend:** Flutter (Dart)
* **Backend:** PHP 8.3 & MySQL 8.0
* **AI Engine:** Python (FastAPI, Scikit-Learn, Pandas) & Google Gemini API
* **Cloud & Infrastructure:** Firebase Cloud Messaging (FCM v1), Google Cloud Platform (GCP)
* **Server Environment:** Apache on Ubuntu Linux 24.04

## ✨ Key Features
* 🤖 **AI-Powered Classification:** Automatically routes complaints to the correct technical department using Machine Learning.
* 📍 **Geolocation Tracking:** Captures the exact coordinates of the user to pinpoint service outages accurately.
* 🔔 **Real-Time Notifications:** Keeps users updated on their complaint status seamlessly via Firebase.
* 📊 **Admin Dashboard:** A responsive web-based control panel for support teams to manage, track, and resolve tickets efficiently.
